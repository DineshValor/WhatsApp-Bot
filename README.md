# Valor Bot

A modern, modular WhatsApp bot built on Baileys (unofficial).

## ⚠️ Disclaimer
This project uses an unofficial WhatsApp Web API.
Use at your own risk. Do NOT use a primary number.

## Requirements
- Node.js >= 18
- WhatsApp account for QR login
